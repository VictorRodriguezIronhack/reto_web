# Reto de The Phone Cave

## Server

### Install

```sh
npm install
```

### Usage

```sh
npm run dev
```

### .env variables needed

- PORT=5005
- ORIGIN=http://localhost:3000

## Client

### Install

```sh
npm install
```

### Usage

```sh
npm run start
```

### .env variables needed

- REACT_APP_API_URL=http://localhost:5005/api
